  • Indexing and serializing C/C++ files
  • DICOM, STL and PDB (Protein Data Bank) file support.
  • Visual UI designs in wxWidgets environment
  • GLSL shader support.
  • CMake support.
  • Makefile support
  • LSP (Language Server Protocol) support.
